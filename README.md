## Build Guide

This project is developed on windows platform, but the build process should be similar on linux or macos.

### Dependencies

1. node.js
2. rust (rustup recommend)
3. pnpm

### Build

```bash
pnpm install
pnpm tauri build
```

## Test

```bash
pnpm vitest --ui
```
